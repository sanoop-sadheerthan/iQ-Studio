# Applications
## Overview

The fastest path from a booted device to a visible inference result — pick a demo, run one command, see it on screen. To customize a demo (swap models, change inputs), see [SDKs](../sdks/README.md). To deploy your own trained model, see [Model Deploy](../model-deploy/README.md).

| Application | Launch command |
| --- | --- |
| [iQS-VLM](./iqs-vlm/README.md) | `iqs-launcher --autotag iqs-ogenie` then `iqs-launcher --autotag iqs-vlm-demo` |
| [iQS-Streampipe](./iqs-streampipe/README.md) | `iqs-launcher --autotag iqs-streampipe` |
| [iQS-YOLOv10n](./iqs-yolov10n/README.md) | `iqs-launcher --autotag iqs-yolov10n` |
